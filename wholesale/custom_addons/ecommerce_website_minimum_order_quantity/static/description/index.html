E-commerce Website Minimum Order Quantity

This Module is designed to set the Minimum order quantity of any products in ecommerce website. You can set minimum order quantity of any product without any hassle and without any code level changes. Install this plugin in your project and follow these basic steps to achieve the desired outcomes.

Set Minimum Order Quantity of products for Customers.

After Installing, you will get the Minimum Order Quantity Field in the admin portal, to add the minimum Amount to set for that particular Product.

Once Done, The work of plugin will start and you don't need to do anything after that. This is end User Screen. The minimum Order wil be reflected to the customer on the Product.

If Customer tries to decrease the minimum quantity of that product, it will show a error message that Minimum quantity for this product is {this}.

However, The customer can Increase the quantity of the product if want till the stock ends.

This Minumum Order Quantity functionality will also work with the cart page as well, throughout the complete Product lifecycle.
